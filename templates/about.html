A propos de UrlShort

UrlShort est un raccourcisseur d'url fait en python. Il a été conçu avec le module flask. Vous pouvez l'utilisez comme un template pour concevoir votre propre raccourcisseur d'url en l'hébergeant sur heroku. UrlShort est un open source. Vous pouvez aussi cloner le repo github et l'héberger directement. Tout ce que vous avez à faire c'est d'ouvrir le fichier details.txt et changer le nom de l'application par votre propre domaine. UrlShort n'utilise aucun autre module en dehors de flask. UrlShort génère des caractères aléatoires à 8 lettres et stocke les stockes pour ne pas faire une répétition.

J'ai conçu UrlShort dans le cadre du developpement web ave flask et celui du Green it. UrlShort est un bon projet pour un début. Vous pouvez trouver les code sur github. Si vous voulez avoir votre propre instance, clonez le repo et initialisez le avec heroku.

Deployement

C'est facile de héberger UrlShort avec le serviceheroku cli. Il ne vous suffit que d'avoir un compte heroku pour le faire.

1. Clonez le repo sur Github

git clone https://github.com/Akon2020/UrlShort.git

2. Connection à heroku cli

heroku login

3. Creation d'une nouvelle application heroku

heroku create "nom_de_l'application"

4. Initialisation et ajout à Git

git init
git add .

5. Le Commit et Le Push

git commit -m "Message du Commit"
git push

6. Patientez qu'Heroku finisse le build et votre application sera prête.
